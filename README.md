## Description

**A recipe app that allows you to look up, add and save liked recipies**

 ## Features

### Must Haves

1. Login / Registration
2. Create recipe
3. View Recipe
4. Home Page
5. Search recipe

### Nice to haves

1. Pantry/Ingredients List
2. Favorites
3. Comments
4. User Profiles
5. Recipe Rating
